# Capstone-EDA-Project-Google-Play-Store-Apps-Reviews-Analysis
Analyzing app trends and user sentiment on Google Play Store to drive business insights using Pandas, Seaborn, and Matplotlib.
Google Play Store Apps & Reviews Analysis
Project Overview
This project aims to analyze the Google Play Store dataset to derive insights into app ratings, pricing strategies, and user sentiment. By cleaning and visualizing the data, we can understand the key factors that contribute to an app's success in a competitive market.

Objectives
Data Cleaning: Handle missing values, duplicate entries, and data type conversions (e.g., converting "Price" to numeric).

Exploratory Data Analysis (EDA): Identify trends across different app categories and genres.

Sentiment Analysis: Analyze user reviews to determine the emotional tone (Positive, Negative, or Neutral) associated with popular apps.

Dataset Description
The analysis utilizes two primary datasets:

data_playstore.csv: Contains details of various applications, including category, rating, size, and installs.

data_user_review.csv: Includes user feedback and sentiment scores for the apps.

Key Features
Visualizing the distribution of app ratings using Seaborn/Matplotlib.

Analyzing the correlation between app size and number of installations.

Comparing free vs. paid apps in terms of user satisfaction.
